Here’s a simple, step-by-step HTML program that clearly demonstrates all your topics in one place.

✅ Complete HTML Program
<!DOCTYPE html>
<html>
<head>
    <title>Lists and Tables Demo</title>
</head>
<body>

<h1>HTML Lists and Tables</h1>

<!-- 1. ORDERED LIST -->
<h2>1. Ordered List</h2>
<ol type="1">
    <li>Wake up</li>
    <li>Go to school</li>
    <li>Study HTML</li>
</ol>

<!-- 2. UNORDERED LIST -->
<h2>2. Unordered List</h2>
<ul type="circle">
    <li>Apple</li>
    <li>Banana</li>
    <li>Mango</li>
</ul>

<!-- 3. DEFINITION LIST -->
<h2>3. Definition List</h2>
<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>

    <dt>CSS</dt>
    <dd>Used for styling web pages</dd>
</dl>


<!-- 4. TABLE WITH COLSPAN AND ROWSPAN -->
<h2>4. Table with Colspan and Rowspan</h2>

<table border="1">
    <tr>
        <th>Name</th>
        <th colspan="2">Marks</th>
    </tr>
    <tr>
        <td rowspan="2">Anand</td>
        <td>Math</td>
        <td>90</td>
    </tr>
    <tr>
        <td>Science</td>
        <td>85</td>
    </tr>
</table>


<!-- 5. TABLE WITH BORDER AND BACKGROUND COLOR -->
<h2>5. Table with Border and Background Color</h2>

<table border="2" bgcolor="lightblue">
    <tr>
        <th>Name</th>
        <th>City</th>
    </tr>
    <tr>
        <td>Anand</td>
        <td>Bangalore</td>
    </tr>
</table>


<!-- 6. TABLE WITH WIDTH, HEIGHT, AND CAPTION -->
<h2>6. Table with Width, Height, Caption</h2>

<table border="1" width="300" height="150">
    <caption>Student Details</caption>
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>Anand</td>
        <td>20</td>
    </tr>
</table>

</body>
</html>
🔍 Step-by-Step Explanation
1. Ordered List (<ol>)
<ol type="1">
Shows numbered list (1, 2, 3)
type can be:
"1" → numbers
"A" → uppercase letters
"a" → lowercase letters
2. Unordered List (<ul>)
<ul type="circle">
Shows bullet points
type:
"disc" (default)
"circle"
"square"
3. Definition List (<dl>)
<dl>
    <dt>Term</dt>
    <dd>Description</dd>
</dl>
<dt> → term
<dd> → definition
4. Table with colspan & rowspan
<th colspan="2">Marks</th>
<td rowspan="2">Anand</td>
colspan="2" → merges 2 columns
rowspan="2" → merges 2 rows

👉 Used in reports, mark sheets

5. Table with Border & Background
<table border="2" bgcolor="lightblue">
border → thickness
bgcolor → background color
6. Table with Width, Height, Caption
<table width="300" height="150">
<caption>Student Details</caption>
width → table width
height → table height
<caption> → table title
