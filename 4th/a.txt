    <optgroup label="Programming">
        <option value="java">Java</option>
        <option value="python">Python</option>
    </optgroup>

    <optgroup label="Web Development">
        <option value="html">HTML</option>
        <option value="css">CSS</option>
    </optgroup>

<h2>3. Input and Textarea</h2>

    Message:<br>

<!-- 4. FIELDSET AND LEGEND -->
<h2>4. Fieldset and Legend</h2>

<form>

<fieldset>
    <legend>Personal Information</legend>

    Name: <input type="text"><br><br>
    Age: <input type="number"><br><br>

</fieldset>

<!-- 5. OUTPUT TAG -->
<h2>5. Output Tag</h2>

<form oninput="result.value = parseInt(a.value) + parseInt(b.value)">

    Number 1: <input type="number" id="a" value="0">
    +
    Number 2: <input type="number" id="b" value="0">

    = 
    <output name="result">0</output>

</form>

</audio>

    <source src="sample.mp4" type="video/mp4">

</video>

🔍 Explanation (Simple & Clear)
1. <button>
type="button" → normal button
onclick → runs JavaScript
2. <datalist> + <option>
<datalist> → gives suggestions
<option> → list values
Used with <input list="id">
3. <select> + <optgroup> + <option>
<select> → dropdown
<optgroup> → group items
<option> → selectable items
4. <input>

Common types:

text → name
email → email validation
password → hidden text
number → numeric input

Attributes:

placeholder → hint text
value → default value
5. <textarea>
Multi-line input
rows, cols → size
6. <fieldset> + <legend>
<fieldset> → groups form fields
<legend> → title of group
7. <output>
Displays calculation result
Used with oninput
8. <audio>
controls → play/pause buttons
<source> → file path
9. <video>
width → size
controls → play options
10. <article>
Represents independent content
Used in blogs, news, posts
🎯 Real-Life Use
Forms → registration/login
Dropdowns → selecting course
Output → calculator
Audio/Video → media player
Article → blog/news content
