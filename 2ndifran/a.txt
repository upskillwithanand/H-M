Alright—let’s build a very simple website layout using <iframe> with:

✅ Header (top)
✅ Footer (bottom)
✅ Left menu
✅ Right panel
✅ Center content

We’ll use multiple HTML files (this is the easiest way to understand iFrames clearly).

📁 Step 1: Main File (index.html)

This file arranges everything.

<!DOCTYPE html>
<html>
<head>
    <title>My iFrame Website</title>
</head>

<frameset rows="80px,*,60px">

    <!-- Header -->
    <frame src="header.html">

    <!-- Middle section -->
    <frameset cols="20%,60%,20%">
        <frame src="left.html">
        <frame src="center.html" name="main">
        <frame src="right.html">
    </frameset>

    <!-- Footer -->
    <frame src="footer.html">

</frameset>

</html>
📁 Step 2: Header (header.html)
<!DOCTYPE html>
<html>
<body style="background-color:lightblue; text-align:center;">
    <h1>My Website Header</h1>
</body>
</html>
📁 Step 3: Footer (footer.html)
<!DOCTYPE html>
<html>
<body style="background-color:lightgray; text-align:center;">
    <h3>My Website Footer</h3>
</body>
</html>
📁 Step 4: Left Menu (left.html)
<!DOCTYPE html>
<html>
<body style="background-color:lightyellow;">
    <h3>Menu</h3>

    <a href="center.html" target="main">Home</a><br><br>
    <a href="about.html" target="main">About</a><br><br>
    <a href="contact.html" target="main">Contact</a>

</body>
</html>
📁 Step 5: Right Panel (right.html)
<!DOCTYPE html>
<html>
<body style="background-color:lightgreen;">
    <h3>Right Side</h3>
    <p>Ads / Info</p>
</body>
</html>
📁 Step 6: Center Content (center.html)
<!DOCTYPE html>
<html>
<body style="text-align:center;">
    <h2>Welcome to Home Page</h2>
    <p>This is center content</p>
</body>
</html>
📁 Step 7: Extra Pages (for menu links)
about.html
<html>
<body>
    <h2>About Page</h2>
</body>
</html>
contact.html
<html>
<body>
    <h2>Contact Page</h2>
</body>
</html>
🔥 Important Note (Very Important)

👉 <frameset> and <frame> are OLD (deprecated in HTML5)
👉 Modern websites use CSS + Flexbox/Grid instead

But for learning:
✔ Helps understand layout
